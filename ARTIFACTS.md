# Artifact Manifest

The repository is intended to share the materials needed to inspect and reproduce the paper-level results without redistributing large model checkpoints.

## Included

- Manuscript source, with compiled PDFs attached as a release asset.
- Paper figures used by the LaTeX manuscript.
- Benchmark source scripts and requirements files.
- Aggregate metrics in CSV form, with the full JSON metrics file attached as a release asset.
- Revision-analysis scripts and outputs for policy ablation, LPIPS scoring, dependency-risk auditing, top-five cross-model ranking, the GCP CUDA L4 detector-inclusive benchmark, L4/T4 operator sweeps, long L4 session benchmarking, bucket model-runtime auditing, and cloud LPIPS scoring.
- Combined JSON/Markdown results and representative grids for the long L4 session benchmark. The raw 174 MB transfer bundle is excluded from Git; the included scripts and summary JSON are sufficient to reproduce and audit the table in the manuscript.
- Human-readable benchmark reports.
- All 200 benchmark `sample_grid.png` outputs, renamed by model, category, and sample, attached as a release asset.
- Citation metadata and SHA-256 checksums for the release ZIP files.
- Hugging Face model-card and detector identifiers in the manuscript bibliography for provenance.

## Excluded

- Diffusion model checkpoint files.
- GroundingDINO, SAM, SAM2, or other external detector weights.
- Full intermediate per-sample artifacts such as every crop, mask, raw local render, and pixel-difference image.
- Local package caches and virtual environments.

The excluded files can be regenerated by running the benchmark with the required external model assets.
